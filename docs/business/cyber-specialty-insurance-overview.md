# Cyber Specialty Insurance Overview

LIAnsureProtect starts with a Cyber MVP for commercial/business customers.

The platform supports two intake paths:

- A customer submits their own cyber insurance application.
- A broker submits an application on behalf of a client company.

## Core Workflow

```text
Company profile
  -> Cyber product selection
  -> Questionnaire
  -> Document upload
  -> Risk scoring
  -> Underwriting review
  -> Quote
  -> Quote acceptance
  -> Policy bind and issue
  -> Claim intake if an incident happens
```

## Cyber Risk Topics

The first questionnaire should model practical cyber underwriting factors:

- Multi-factor authentication
- Endpoint detection and response
- Backup practices
- Incident response plan
- Security awareness training
- Prior cyber incidents
- Industry risk
- Revenue size
- Sensitive data handling
- Cloud dependency

## Product Expansion

The first build focuses on Cyber Liability. The product catalog should be flexible enough to add Tech E&O, MPL, and Multimedia Liability later.

Do not hard-code the app as if only one insurance product can ever exist.
